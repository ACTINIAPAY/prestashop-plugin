# Установка модуля оплаты Actinia для Prestahop 1.6
Для установки модуля приема платежей необходимо скачать архив с нашего репозитория Github. 
  
Распаковать архив в папку **/modules/actinia** в корне сайта.  
Зайдите в админ-часть. **Модули -> Модули и Сервисы**.  
Найдите в списке " Actinia ... " и нажмите "Установить".  
После установки нажмите "Настройка" и заполните необходимые поля. Сохраните.
